# 2026-06-26 工作记录

## CalMac 项目全面修复与优化

在 2026-06-25 创建项目的基础上，完成 12 项修复与优化，前端 `vue-tsc` 类型检查通过，后端 `cargo check` 编译通过。

### 一、Bug 修复

#### 1. 时区日期错位（影响最广）
- 新建 `src/utils/date.ts`，提供 `toLocalDateKey` / `todayKey` / `isoToLocalDateKey` 等本地时区安全函数
- 替换所有 `new Date().toISOString().split("T")[0]` 用法（calendar store、todo store、TodoPanel）
- `getEventsForDate` 改用本地时间解析，避免 UTC 跨天错位

#### 2. 2026 节假日数据缺失
- `src/utils/holidays.ts` 补充 `HOLIDAYS_2026` / `WORK_DAYS_2026`（参考国务院公告）
- 重构查询逻辑为 `HOLIDAYS_BY_YEAR` / `WORK_DAYS_BY_YEAR` map，新增年份只需加一条

#### 3. 待办 sort_order 恒为 0
- `todo_service.rs` 新建时按同组（global / daily+date）查 `MAX(sort_order) + 1`
- 新增 `next_sort_order` 辅助函数

#### 4. 事件创建/编辑 UI 缺失
- 新建 `src/components/Calendar/EventForm.vue` 弹窗，支持标题/全天/起止时间/提醒/颜色/备注
- 接入 MonthView：点击格子新建、点击事件编辑、右键菜单「新建日程」
- 支持创建、编辑、删除全流程，含动画过渡

#### 5. 右键菜单越界
- MonthView 用 `menuRef` + `nextTick` 测量菜单尺寸，超出视口翻转定位

### 二、安全与健壮性

#### 6. CSP 安全策略
- `tauri.conf.json` 将 `csp: null` 改为白名单：限制 default-src 'self'，放开 img/style/script/connect 所需来源

#### 7. 数据库版本化迁移
- `database.rs` 引入 `schema_meta` 版本表 + `migrate` 函数
- 当前 version=2，v2 通过 `ALTER TABLE` 给 events 补 `remind_minutes` 列
- `column_exists` 辅助函数避免重复 ALTER

#### 8. 类型对齐与枚举校验
- Rust `models/mod.rs` 新增 `TodoType` / `Priority` 枚举（serde rename_all lowercase）
- 写入时 serde 拒绝非法值，读取时容错回退
- `Todo` 结构体字段类型对齐枚举

### 三、性能优化

#### 9. 事件查询按月范围过滤
- 后端新增 `get_events_by_range(start, end)` 命令，SQL `WHERE start_time >= ?1 AND start_time < ?2`
- 前端 calendar store 新增 `loadMonth(date)`，按视图覆盖范围（前后扩展到周首尾）查询
- App.vue `watch(currentDate)` 自动重载

#### 10. 事件按日分组缓存
- calendar store 新增 `eventsByDate` computed（`Map<dateKey, Event[]>`），O(n) 一次分组
- `getEventsForDate` 改为 O(1) 查表

### 四、新功能

#### 11. 提醒功能
- Event 新增 `remind_minutes` 字段（`Option<i32>`）
- 新建 `services/reminder.rs`：tokio 后台循环每 60 秒检查
  - 计算提醒时刻 `start_time - remind_minutes`，命中窗口且未通知过则发系统通知
  - `HashSet` 记录已通知 ID，过期 24h 自动清理
- 接入 `tauri-plugin-notification`，`main.rs` 启动时 spawn 调度循环
- 新建 `capabilities/default.json` 配置 notification 权限

### 五、代码清理
- 删除未被引用的死代码 `src/components/Common/Sidebar.vue`
- 更新 `services/mod.rs` 注册 reminder 模块
- 更新 `commands/calendar.rs` 注册 `get_events_by_range`
