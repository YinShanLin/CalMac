# 2026-06-25 工作记录

## CalMac 项目创建

### 项目概述
- **项目名称**: CalMac (macOS 日历应用)
- **技术栈**: Tauri 2.0 + Vue 3 + TypeScript + SQLite
- **目标平台**: macOS

### 已完成工作

#### 1. 项目结构创建
- 创建了完整的项目目录结构
- 配置了前端 (Vue 3 + Vite) 和后端 (Rust + Tauri) 的基础文件

#### 2. 前端文件
- `package.json` - Node.js 依赖配置
- `vite.config.ts` - Vite 构建配置
- `tsconfig.json` - TypeScript 配置
- `index.html` - 入口 HTML
- `src/main.ts` - Vue 应用入口
- `src/App.vue` - 根组件
- `src/types/event.ts` - 事件类型定义
- `src/stores/calendar.ts` - Pinia 状态管理
- `src/components/Common/Header.vue` - 头部导航组件
- `src/components/Common/Sidebar.vue` - 侧边栏组件
- `src/components/Calendar/MonthView.vue` - 月视图组件
- `src/components/Calendar/EventItem.vue` - 事件项组件

#### 3. 后端文件
- `src-tauri/Cargo.toml` - Rust 依赖配置
- `src-tauri/tauri.conf.json` - Tauri 配置
- `src-tauri/build.rs` - 构建脚本
- `src-tauri/src/main.rs` - Rust 入口
- `src-tauri/src/models/mod.rs` - 数据模型
- `src-tauri/src/services/database.rs` - 数据库服务
- `src-tauri/src/services/event_service.rs` - 事件服务
- `src-tauri/src/commands/calendar.rs` - Tauri 命令
